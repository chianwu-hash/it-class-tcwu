# 三年級資訊課教案｜Week 08

- 週次：第 08 週
- 主題：校務行政系統改密碼
- 本稿定位：本節主任務只放「改密碼」，其餘時間回前幾週中打闖關
- 節數：1 節（40 分鐘）
- 任務規劃現況：
  1. 任務一：校務行政系統改密碼
  2. 緩衝任務：回前幾週中打闖關

## 任務一教學目標
1. 能知道今天改密碼是因為系統要更安全。
2. 能知道親師生平台的密碼和校務行政系統是連在一起的。
3. 能跟著老師找到 `個人資料/密碼修改` 的位置。
4. 能在老師協助下完成一次新密碼設定。
5. 能說出至少 2 種不能拿來當密碼的資料。
6. 能記住自己改好的新密碼，不只記得舊密碼。

## 任務一核心概念
1. 今天改密碼不是多一個新帳號，而是把平常會用到的密碼改得更安全。
2. 因為教育局要求，所有學生都要把太簡單、太好猜的弱密碼改掉。
3. 如果弱密碼沒有改好，之後可能會不能用。
4. 很多學生現在用的舊密碼，可能就是身分證字號這種太容易被猜到的資料，所以今天一定要改。
5. 密碼不能用生日、身分證字號、自己名字、帳號這些太容易被猜到的資料。
6. 看不懂、找不到、不能確定時，要先舉手，不要亂按。

## 任務一時間配置
- 任務一總長：35 分鐘
- 點名與環境整理：5 分鐘
- 先登入教學網站與等待任務說明：3 分鐘
- 任務說明與規則建立：4 分鐘
- 校務行政系統登入與教師救援：8 分鐘
- 教師示範與全班同步改密碼：12 分鐘
- 學生操作：5 分鐘
- 收整與確認：3 分鐘
- 緩衝任務：若提早完成，回前幾週中打闖關

## 任務一教學流程
1. 點名與環境整理（5 分鐘）
   - 先完成點名。
   - 確認學生電腦和網路正常。
   - 提醒學生先整理好桌面和座位，再開始今天的任務。

2. 先登入教學網站與等待任務說明（3 分鐘）
   - 先請學生登入教學網站。
   - 告訴學生今天如果先完成校務行政系統登入，就先回到前幾週的中打闖關等待。
   - 先講清楚等待規則：
     - 先登入成功的人，先去中打闖關
     - 老師說停，就要立刻停下來看投影
     - 還沒跟上全班步驟前，不可以自己亂按校務行政系統

3. 任務說明與規則建立（4 分鐘）
   - 老師先用白話說明：
     - 今天不是自由上網時間。
     - 今天要做一件重要事情：把密碼改安全一點。
     - 因為教育局要求，所有學生都要把弱密碼改掉。
     - 如果沒有改好，之後可能會不能用。
     - 很多同學原本用的可能就是身分證字號這種舊密碼，今天要把它換掉。
   - 先講清楚學生最需要知道的事：
     - 你們平常比較常用的是親師生平台。
     - 但是親師生平台的密碼，和校務行政系統是連在一起的。
     - 所以今天改的密碼，會影響你之後平常登入。
   - 老師統整成一句話：
     - `今天改好密碼，以後登入才不會卡住。`
     - `而且一定要改成自己記得住的密碼。`
   - 先講不能拿來當密碼的資料：
     - 生日
     - 身分證字號
     - 自己名字
     - 帳號
   - 再講今天統一用的簡單密碼公式：
     - `小寫英文 + 數字`
   - 範例：
     - `tiger35`
     - `panda42`

4. 校務行政系統登入與教師救援（8 分鐘）
   - 全班一起進入 `https://esa.ntpc.edu.tw`
   - 全班一起點首頁左邊的 `登入(Login)`
   - 學生開始輸入原本帳號和原本密碼
   - 老師優先協助還沒成功登入的學生：
     - 大小寫輸錯
     - 把帳號打進密碼欄
     - 忘記原本密碼
   - 已成功登入的學生，先回到教學網站做中打闖關等待，不先自己往下改密碼
   - 等大多數學生都已成功登入後，老師再統一把全班帶回改密碼流程

5. 教師示範與全班同步改密碼（8 分鐘）
   - 投影示範登入後要看右上角自己的名字
   - 示範：
     - 先點右上角名字
     - 再點 `個人資料/密碼修改`
   - 進入頁面後，再示範在 `自訂帳號與密碼` 區塊點 `變更密碼`
   - 告訴學生要填的 4 個地方：
     - 舊密碼
     - 新密碼
     - 再打一次新密碼
     - 驗證碼
   - 先把最容易弄錯的地方講白：
     - 第一格 `舊密碼` 不是帳號，是你原本的密碼
     - 第二格 `新密碼` 是你今天要換的新密碼
     - 第三格 `再打一次新密碼` 不是再想一個新的，是把第二格再打一遍
   - 可帶全班一起念一次：
     - `第一格是舊密碼`
     - `第二格是新密碼`
     - `第三格是把新密碼再打一遍`
   - 用中年級能懂的方式再講一次規則：
     - 要夠長
     - 要有英文
     - 要有數字
   - 老師直接統整：
     - `最簡單就是先想一個小寫英文，再加上數字。`
   - 再提醒：
     - 兩次新密碼要打一樣
     - 驗證碼要看清楚
     - 沒成功前不要亂關視窗
     - 不要亂設一個等一下就忘記的密碼

6. 學生操作：完成改密碼（9 分鐘）
   - 學生跟著剛剛全班步驟繼續做：
     1. 進入校務行政系統
     2. 點登入
     3. 輸入原本帳號密碼
     4. 點右上角名字
     5. 點 `個人資料/密碼修改`
     6. 點 `變更密碼`
     7. 輸入舊密碼、新密碼、確認新密碼、驗證碼
     8. 按 `儲存`
   - 老師巡迴優先幫忙：
     - 找不到右上角名字的學生
     - 找不到 `變更密碼` 的學生
     - 新密碼規則不合的學生
     - 驗證碼一直失敗的學生
     - 忘記原本密碼的學生
   - 先完成改密碼的學生，可回到教學網站繼續中打闖關。

7. 收整與確認（3 分鐘）
   - 請學生舉手回報是否已完成改密碼。
   - 再問一次今天最重要的提醒：
     - 密碼不能用什麼？
     - 如果看不懂要怎麼辦？
     - 新密碼要改成什麼樣子？
   - 老師收尾統整：
     - `記住新密碼，不要下次又只記得舊密碼。`
     - `一定要改成自己記得住的密碼。`
   - 先完成改密碼的學生，可回到前幾週的中打闖關繼續練習。

## 任務一學習任務
1. 能進入校務行政系統登入畫面。
2. 能找到 `個人資料/密碼修改`。
3. 能在老師提醒下完成新密碼設定。
4. 能說出至少 2 項不能放進密碼的資料。

## 任務一過關條件
1. 成功登入校務行政系統。
2. 成功找到 `變更密碼` 的位置。
3. 完成一次新密碼設定。
4. 能口頭說出：
   - 不要用生日
   - 不要用身分證字號
   - 不要用自己名字
   - 不要用帳號
   其中任 2 項。
5. 先完成的學生，可自行回前幾週中打闖關頁面繼續練習。

## 任務一教師提醒
- 三年級學生對「帳號連動」比較抽象，要反覆用白話講：`今天改這個，以後親師生平台也會一起用到。`
- 不要一次講太多資安名詞，只抓「太好猜的密碼不行」這個重點。
- 三年級這節課不要讓學生在老師救援別人時，自己亂往下按校務行政系統；先登入成功的人，先回教學網站做中打闖關等待。
- 對三年級來說，最大卡點通常是找不到右上角名字，以及不知道還要再點一次 `變更密碼`。
- 另一個大卡點是把 `舊密碼` 誤填成帳號，或把第三格當成再想一個新密碼，這段一定要先帶全班口頭講過一次。
- 還要注意少數學生連用舊密碼登入都會卡住，常見原因是大小寫輸錯、帳號密碼欄位弄反，或原本密碼本來就記不清楚。
- 若學生很難想密碼，不要臨場發明很多版本，統一回到本節公式即可。
- 三年級先以 `小寫英文 + 數字` 為主，若個別學生真的無法通過，再個別協助調整。
- 一定要反覆提醒：不是隨便改一組能過就好，而是要改成自己記得住的密碼；否則老師幫忙還原後，又得再改一次，會一直重來。
- 不要讓學生公開念出自己的新密碼，也不要要求整個班一起展示完整密碼。
- 若學生卡住，不要只說「再試一次」，要直接指出他現在卡在哪一步。
- 本節重點是全班順利完成改密碼，不再額外塞入新的行政任務。
- 先完成的學生回前幾週中打闖關，讓老師能把注意力留給還在改密碼的學生。

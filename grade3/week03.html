滑鼠鍵盤魔法師

要成為電腦高手，一定要學會這三個超強的魔法！

影印機魔法 (複製)

Ctrl + C

把選起來的字，偷偷印一份在電腦的腦袋裡。

黏土魔法 (貼上)

Ctrl + V

把剛剛印好的字，像黏土一樣貼在你想貼的地方。

時光倒流 (復原)

Ctrl + Z

哎呀！不小心貼錯了？按這個就能回到上一步！

魔鬼訓練營

小關卡：通關密語大挑戰！

老師特別提醒：選取是最重要的一步！

如果沒有把字變成「藍色底」，電腦就不知道你要複製什麼魔法喔！請記住三個選取小秘訣：

  • 起點最重要： 滑鼠游標一定要準確放在「第一個字」的前面，再按住左鍵。
  • 拖錯不放手： 萬一拖曳時選歪了，千萬不要放開左鍵！繼續移動滑鼠找回正確位置再放開就好。
  • 點空白重來： 如果真的選錯並放開了，只要在旁邊空白的地方點一下左鍵，就能取消選取重新再來。

第一步：按住滑鼠左鍵，將下面的「多行」藍色文字全部反白，然後按下 Ctrl + C 複製。

# Week 12 Music Pack

This folder contains teacher-selected Pixabay Music tracks for Grade 6 Week 12.

Use case:
- Canva graduation video background music
- Classroom practice and school-internal presentation
- Not intended for public YouTube upload without checking each track page again

Deployment:
- The formal playback source should be Cloudinary, not local MP3 files.
- Cloudinary URLs are listed in `cloudinary-music.json`.
- Local MP3 files are kept as source backup and can be removed before deployment if repo size becomes a concern.

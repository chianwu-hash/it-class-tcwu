# 校內段考卷參考區

這個資料夾用來保存校內歷屆段考卷，作為命題風格與難度校準的參考來源。

目前規劃只收：
- 六年級 `113下`
- 六年級 `114上`

使用原則：
- 校內考卷只用來觀察命題風格、難度、措辭與常見題型。
- 校內考卷不能凌駕教材、教師手冊、考試範圍與題庫品質規範。
- 若歷屆考卷出現疑似超綱、錯題、後設題或不良干擾項，不可直接沿用。

建議放檔方式：
- 原始檔直接放進各科的 `raw/` 資料夾。
- 可接受格式：`pdf`、`jpg`、`jpeg`、`png`、`docx`。
- 先不用強制改檔名；放入後再由分析流程統一整理。

後續分析順序：
1. 先依學期與科目讀取原始考卷
2. 整理出題型、難度、常見措辭與干擾項風格
3. 標記不該沿用的舊題特徵
4. 才把結論納入角色定位與提示詞

# 六年級 114 上自然命題風格分析

## 整體印象

114 上自然題目比 113 下更明顯走「大題組整合」路線，除了是非與選擇，還有大量依圖作答、填空與資料判讀。

## 題型結構

- 期中卷有熱傳、水循環等生活科學題組。
- 期末卷則涵蓋地形、礦物、地震、地磁、電磁鐵等多個單元。
- 除了一般選擇題，也常出現圈選、填代號、畫流動方向、根據實驗表推論等任務型題目。

## 明顯風格

- 題目非常依賴圖示、裝置圖與資料表。
- 會用長篇生活情境包住科學概念，例如煮湯、循環扇、村落河道、地震報告。
- 除了記憶事實，也要求推論因果、解釋現象、讀懂實驗結果。
- 本校自然卷願意讓學生在同一大題裡處理多小題，這點跟一般短題型題庫差異很大。

## 對後續出題的啟示

- 自然題庫要保留「題組感」，不能全部拆成孤立小題。
- 若轉成純文字題庫，要小心圖示資訊流失；必要時應回頭參照原卷。
- 角色 prompt 應明講：自然科題目要從實驗、現象與資料出發，而不是只問定義。

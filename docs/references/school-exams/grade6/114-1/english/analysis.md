# 六年級 114 上英語命題風格分析

## 整體印象

114 上英語和 113 下高度一致，整份卷面仍維持聽力加讀寫的雙主軸，風格穩定、明確，幾乎就是本校英語命題模板。

## 題型結構

- 固定分成 `Part I. Listening` 與 `Part II. Reading and Writing`。
- 聽力題型包含聽音選字、聽圖勾選、聽後配對、聽後選答。
- 讀寫題型通常包含看圖寫字、句型理解、短文閱讀、時間地點配對與情境問答。

## 明顯風格

- 指令雙語並列，學生很清楚每題要做什麼。
- 核心主題圍繞日常生活，像時間、地點、休閒活動、旅遊、人物過去和現在等。
- 英語卷不偏艱深文法術語，而是偏情境與句意理解。
- 選項設計傾向簡潔明確，不太做曖昧或雙關。

## 對後續出題的啟示

- 本校英語應維持生活化、情境化、技能導向，不要滑向抽象文法測驗。
- 若未來要做成題庫格式，也應保留「這題對應口說/聽力/情境理解」的出題感。
- 英語角色 prompt 應明講：像國小英語老師出題，不像文法分析師出題。

# 六年級 114 上國語命題風格分析

## 整體印象

114 上國語延續本校一貫風格，仍然重視國字注音、詞語辨析、成語與閱讀理解，但語文知識題更明確，也更常直接考修辭與設問技巧。

## 題型結構

- 期中卷先考國字注音，再進入大量選擇題。
- 期末卷常見寫國字或注音、成語練習、改錯字，再接閱讀與選擇題。
- 題目分區清楚，配分明確，偏好傳統段考版型。

## 明顯風格

- 除了字詞基本功，也會考修辭、文章主旨、人物關係、課文訊息與語句理解。
- 會直接問「哪個選項使用了相同寫作技巧」、「哪個沒有頂真修辭」這類語文知識題。
- 國語卷接受一定比例的術語型語文題，這是本校實際風格的一部分。
- 期末仍會結合課文與情境，但不會放棄基本功題。

## 對後續出題的啟示

- 若我們要貼近本校國語風格，不能把所有題目都改成素養敘事題。
- 國語可以保留語文知識與修辭判讀，但題目仍應服務於學生語文能力，而不是命題技巧分析。
- 本校國語命題風格比一般想像中更願意考「語文方法」，這點角色 prompt 要如實反映。

# 六年級 114 上社會命題風格分析

## 整體印象

114 上社會與 113 下一致，依舊是「大量是非題 + 選擇題 + 少量題組」的穩定段考風格，偏重史實、公民、政策與社會變遷理解。

## 題型結構

- 期中卷先放大量是非題，再接選擇題。
- 期末卷也是先是非再選擇，題目常跨歷史、經濟、人口、區域發展與全球互動。
- 題幹多為完整敘述句，常要求學生找出錯誤或不可能的敘述。

## 明顯風格

- 很重視年代、制度、政策名稱與其影響之間的對應。
- 選項常設計成「看起來都合理，但只有一個細節不對」。
- 雖然有生活情境，例如搭火車、人口遷移、地方生活圈，但本質仍是教材概念檢核。
- 社會卷特別需要學生分辨名詞、時序與因果關係。

## 對後續出題的啟示

- 本校社會題庫可以維持較高比例的判斷題與敘述型選擇題。
- 不能只做大方向主旨題，必須保留教材細節與制度脈絡。
- 社會角色 prompt 應該加入「善於把史實與制度影響放進完整敘述句」這個特徵。

# 六年級校內命題風格總結

這份摘要根據 `113下` 與 `114上` 共 20 份校內段考卷整理，目的是讓後續角色定位與提示詞有「本校實際命題風格」可依據。

## 全科共通特徵

- 題目結構清楚，幾乎都會先分大題、標示配分，再進入作答。
- 題目整體偏教材導向，不走開放式素養作文型評量。
- 很重視基礎能力與教材細節，不避諱考定義、名詞、年代、語詞、修辭或公式。
- 真正的難度通常來自細節辨識、多步驟整理與情境套用，不是靠文字陷阱。
- 很多題目會先給生活情境或完整敘述句，再要求學生判斷概念是否正確。

## 各科特徵

### 國語

- 重視國字注音、改錯字、成語、閱讀理解、修辭與寫作技巧。
- 本校國語確實會考語文術語，但仍屬語文能力範圍，不是命題後設知識。
- 題型結構本身也很穩定，不能只模仿「考課文理解」，還要保留本校常見的基礎語文題型配置。

### 英語

- 穩定採用聽力加讀寫雙主軸。
- 偏生活化、情境化、發音與句意理解，不走艱深文法分析。

### 數學

- 少量選擇題搭配大量填充、計算與題組。
- 重視單位換算、圖表判讀、速率與可驗算的應用情境。

### 自然

- 題目常結合實驗、裝置、圖示與真實現象。
- 題組比例高，學生需要讀懂情境後再逐題推論。

### 社會

- 大量使用是非題與敘述型選擇題。
- 特別重視史實、制度、政策與社會影響的對應。

## 對角色定位最重要的結論

- 角色不能只是泛化的「國小老師」，而要更像熟悉本校段考結構的資深任課老師。
- 角色也不能只站在評量專家視角，否則容易寫出不像本校考卷的後設題。
- 最穩的定位應是：
  `熟悉台灣 108 課綱、教材版本與本校六年級段考風格的資深任課老師，並具備評量設計能力。`

## 目前仍要保留的限制

- 校內考卷只是風格校準來源，不可壓過教材、教師手冊與本專案品質規範。
- 若歷屆考卷出現疑似後設題、超綱題或不良誘答項，不應直接沿用。
- 文字抽取已足夠支援風格分析，但圖示細節仍要在必要時回看 `raw/` 原卷。

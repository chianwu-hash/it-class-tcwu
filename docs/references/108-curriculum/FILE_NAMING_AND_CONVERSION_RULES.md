# 檔名與轉檔規則

## 原則

- 每份參考文件都要保留官方原始檔與可搜尋的工作檔。
- 正式原始檔優先使用 `PDF`；若官方頁面本身有錯誤或只提供其他格式，保留官方原始格式並在 `notes.md` 註明原因。
- 工作檔一律轉成 `UTF-8` 編碼的 `TXT`。
- 工作時先讀 `TXT`，遇到版面、表格或語句疑義時回頭核對原始檔。

## 固定命名

### 總綱

- `general/curriculum-framework.pdf`
- `general/curriculum-framework.txt`

### 各科領綱

- `subjects/chinese/curriculum-outline.pdf`
- `subjects/chinese/curriculum-outline.txt`
- `subjects/english/curriculum-outline.pdf`
- `subjects/english/curriculum-outline.txt`
- `subjects/math/curriculum-outline.odt`
- `subjects/math/curriculum-outline.txt`
- `subjects/science/curriculum-outline.pdf`
- `subjects/science/curriculum-outline.txt`
- `subjects/social/curriculum-outline.pdf`
- `subjects/social/curriculum-outline.txt`

### 各科課程手冊

- `subjects/chinese/curriculum-manual.pdf`
- `subjects/chinese/curriculum-manual.txt`
- `subjects/english/curriculum-manual.pdf`
- `subjects/english/curriculum-manual.txt`
- `subjects/math/curriculum-manual.pdf`
- `subjects/math/curriculum-manual.txt`

- `subjects/science/curriculum-manual.txt`

- `subjects/social/curriculum-manual.txt`

### 補充摘要

- `subjects/chinese/notes.md`
- `subjects/english/notes.md`
- `subjects/math/notes.md`
- `subjects/science/notes.md`
- `subjects/social/notes.md`

## 轉檔要求

- PDF 以可擷取文字為主，先直接抽字，不先做 OCR。
- ODT 直接保留原始檔，再轉成 `TXT`。
- `TXT` 要保留基本段落結構，至少能讀出標題、目次、學習重點與附錄名稱。
- 若發現原始來源連結有誤、檔案內容不符、或抽字品質不足，必須在對應 `notes.md` 記錄。

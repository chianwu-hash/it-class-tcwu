# 出題前參考順序

六年級題庫命題時，參考順序固定如下，不能跳過前兩層直接拿課綱生題。

## 1. 教材與教師手冊

先確認實際使用的版本、課次、閱讀篇章、例題與教學重點。段考題應該優先對應這一層。

## 2. 實際考試範圍

確認這次命題是否只考某幾課、某幾單元，或有老師另外指定不考與加強的內容。這一層決定題目可不可以出。

## 3. 108 課綱領域綱要

用來檢查年段能力、學習表現、學習內容與核心素養方向，避免出到失焦題、後設題或超綱題。

## 4. 108 課程手冊

用來補充理解該領域在 108 課綱下的教學取向、評量原則與常見命題提醒，不直接取代教材。

## 出題提醒

- 角色定位要站在熟悉台灣 108 課綱與教材版本的資深任課老師。
- 題目要測學生的學科內容，不測命題技巧、教學術語或流程知識。
- 難題要來自更深的理解、比較、推論或應用，不是來自抽象措辭或後設敘述。
- 誘答項要反映學生真實可能犯的錯，不可荒謬，也不可讓正確答案過度突出。

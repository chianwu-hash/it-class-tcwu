# 108 課綱參考區

這個資料夾用來保存六年級題庫命題會參照的 108 課綱正式文件與工作檔，目的是讓後續出題、複查與 Gemini 驗證都能回到同一套上位依據，而不是靠印象判斷。

命題參考順序固定如下：
1. 教材與教師手冊
2. 實際版本、單元與考試範圍
3. 108 課綱領域綱要
4. 108 課程手冊

目前收錄的範圍：
- `general/`：總綱
- `subjects/chinese/`：國語文領綱、課程手冊、摘要
- `subjects/english/`：英語文領綱、課程手冊、摘要
- `subjects/math/`：數學領綱、課程手冊、摘要
- `subjects/science/`：自然科學領綱、課程手冊、摘要
- `subjects/social/`：社會領綱、課程手冊、摘要

使用原則：
- 正式原始檔優先保留在 `PDF` 或官方原始格式。
- 工作時優先查 `TXT`，但有疑義時一定回頭核對原始檔。
- `notes.md` 用來記錄來源、下載日期、命題重點與特別注意事項。
- 題目不可直接照抄課綱句子，而要轉成符合國小段考情境的學生題。

例外情況：
- 數學領域在官方課綱頁面上的 `PDF` 連結目前誤指向課程手冊檔案。為了保留正確內容，本專案改採同列官方 `ODT` 作為數學領綱原始檔，再轉成 `TXT` 供工作使用。

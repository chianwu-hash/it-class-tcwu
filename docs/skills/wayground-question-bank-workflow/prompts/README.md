# Question-Bank Prompts

這個資料夾保存六年級題庫重製時要使用的核心提示詞。

結構分成兩層：
- `common/`：所有科目共用的母提示詞
- `subjects/`：五科分科提示詞

使用方式：
1. 先套用共用母提示詞
2. 再疊加對應科目的分科提示詞
3. 最後再補上本次教材範圍、題數、難度分配與輸出格式

這些提示詞的依據來自：
- 本專案題庫工作流與品質規格
- 108 課綱與課程手冊參考區
- 六年級 `113下`、`114上` 校內段考卷命題風格分析

注意：
- 這些提示詞是命題與審題的行為規範，不是直接給學生看的內容。
- 若本次教材範圍和校內舊卷風格衝突，仍以教材與本次考試範圍優先。

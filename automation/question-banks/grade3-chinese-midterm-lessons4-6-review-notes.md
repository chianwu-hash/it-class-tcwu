## Q23（困難）：比較第4課和第5課的故事結構

**題目：** 比較第4課「魯班發明鋸子」和第5課「皮爾森發明地鐵」的故事結構，下列哪一項最合理？

**答案：B**

### 干擾項說明

- **A**「兩課都是在記錄發明家的日常生活，沒有特別的起因和解決過程」
  學生猶豫的理由：兩課確實都在描寫真實人物的生活，部分學生把「記錄人物」和「記錄日常生活」混淆；沒有注意到「蓋王宮需要木頭」和「倫敦交通擁擠」是明確的起因。
  錯誤所在：兩課都有清楚的背景→起因→經過→結果，A 說「沒有」是錯的。

- **C**「第4課有完整的故事結構，第5課只寫了結果，沒有過程」
  學生猶豫的理由：第5課的過程（皮爾森說服議員、不分日夜設計）篇幅比第4課短，學生可能印象不深，誤以為沒有寫到過程。
  錯誤所在：第5課有完整的過程描述（皮爾森如何提議、遭質疑、堅持設計），並非只寫結果。

- **D**「兩課都以議員或官員為主角，重點是描寫他們如何做決定」
  學生猶豫的理由：第5課確實出現議員，可能讓學生誤以為議員是主角。
  錯誤所在：魯班和皮爾森才是兩課的主角；議員只是烘托皮爾森性格的配角。

## Q24（困難）：「大家都叫他工匠之祖」的結尾作用

**題目：** 第4課中，魯班的故事最後說「大家都叫他工匠之祖」，這個結尾有什麼作用？

**答案：B**

### 干擾項說明

- **A**「只是說明魯班的職業名稱」
  學生猶豫的理由：「工匠之祖」包含「工匠」二字，學生可能把「之祖」的榮銜誤讀為單純的職業說明。
  錯誤所在：「之祖」是表達對某一領域奠基者的尊稱，並非職業分類。

- **C**「提醒讀者要記住魯班的名字」
  學生猶豫的理由：課文結尾點名「魯班」，學生可能以為作者目的只是強調名字。
  錯誤所在：結尾的重點是「工匠之祖」這個稱號所代表的歷史定位和後人的肯定，而不是提醒名字。

- **D**「告訴讀者春秋時代的歷史背景」
  學生猶豫的理由：春秋時代是開頭的背景資訊，學生可能將結尾的功能與開頭的功能搞混。
  錯誤所在：結尾的功能是「收束」，把全文的發明歷程和人物價值做一個總結，而非補充歷史背景。

## Q25（困難）：議員與皮爾森對比的作用

**題目：** 第5課中，作者安排「議員們提出疑問」和「皮爾森努力說服」這兩組人物的對比，最主要的作用是什麼？

**答案：C**

### 干擾項說明

- **A**「讓讀者知道議員是壞人」
  學生猶豫的理由：議員一開始反對，學生可能把「阻礙主角」等同於「壞人」。
  錯誤所在：議員提出安全和費用的疑慮是合理的保守態度，課文並無負面評價；對比是為了凸顯皮爾森的性格，不是要批判議員。

- **B**「說明蓋地鐵很困難，應該放棄」
  學生猶豫的理由：議員列出許多困難，讓學生感受到阻礙，可能誤以為重點是「困難難以克服」。
  錯誤所在：課文的結論是皮爾森克服了困難，地鐵成功建成；重點是「堅持可以克服阻力」，而非「應該放棄」。

- **D**「說明所有政府官員都不喜歡新發明」
  學生猶豫的理由：課文中議員確實提出反對，學生可能將個別情境過度推廣為普遍結論。
  錯誤所在：課文只描述這一次的議員反應，並未對「所有官員」做出概括性評論。

## Q26（困難）：「帶人類通往更便利的新世界」的修辭

**題目：** 第6課最後寫「人類的想像力無限，我相信透過網路和科技，這個神奇密碼將會帶人類通往更便利的新世界！」這句話用了什麼修辭手法？

**答案：B**

### 干擾項說明

- **A**「比喻，把密碼比喻成地圖」
  學生猶豫的理由：「通往」讓人聯想到路或地圖，句中又有「新世界」這類比喻性語詞，學生可能誤判為比喻。
  錯誤所在：比喻需要明確的「喻體＝被比較的對象」；這句話沒有把密碼直接比作地圖，而是賦予密碼「帶領人類」的主動能力，是擬人。

- **C**「設問，提出問題讓讀者思考」
  學生猶豫的理由：設問是第5課的修辭重點，學生可能把「我相信……」句型誤讀成反問句。
  錯誤所在：這句話是肯定句＋感嘆號，不含問句語氣；設問需要先自問再自答，本句無此結構。

- **D**「類疊，重複相同字詞加強語氣」
  學生猶豫的理由：類疊是第5課的修辭類型之一，學生學過後可能會「到處套用」。
  錯誤所在：類疊需要重複出現相同或相近的字詞；本句沒有明顯的重複字詞組合。

## Q27（困難）：反駁「皮爾森只是幸運」的論點

**題目：** 如果有人說：「皮爾森只是幸運，剛好看到田鼠而已，換成別人也會想到地鐵。」下列哪個回應最符合課文？

**答案：B**

### 干擾項說明

- **A**「對，因為田鼠在地下跑這件事誰都看得到」
  學生猶豫的理由：看到田鼠挖洞確實是生活中可能發生的事，學生可能覺得這個「觀察機會」人人都有，所以支持「誰都可以」的說法。
  錯誤所在：課文強調皮爾森是長期苦思交通問題的律師，才能在看到田鼠時立刻連結到解決辦法；「看得到」≠「想得到」，沒有積累的思考就沒有靈感。

- **C**「對，因為課文沒有說皮爾森特別聰明」
  學生猶豫的理由：課文確實沒有明說皮爾森「天才」或「特別聰明」，學生可能以為沒有強調聰明就等於承認了「只是運氣」。
  錯誤所在：課文強調的不是聰明，而是「長期關注問題＋仔細觀察＋堅持不懈」；這些是比天資更重要的成功條件，與「幸運」的說法相矛盾。

- **D**「不對，因為皮爾森是律師，比別人更聰明」
  學生猶豫的理由：律師是高學歷職業，學生可能覺得「律師一定比別人聰明」，因此認為這是好的反駁理由。
  錯誤所在：課文沒有暗示律師比別人聰明；反駁的關鍵是皮爾森長期思考交通問題的積累，而非職業帶來的智力優勢。

## Q28（困難）：三課共同傳達的想法

**題目：** 從第4課到第6課，三課共同傳達的想法最接近下列哪一項？

**答案：B**

### 干擾項說明

- **A**「只要努力工作，不需要觀察和思考」
  學生猶豫的理由：三課的主角都很努力（魯班砍木頭、皮爾森不分日夜設計、QR Code 發明者持續研究），學生可能只記住「努力」而忽略「觀察」。
  錯誤所在：三課都明確強調靈感來自觀察（草葉尖齒、田鼠挖洞、生活應用場景），缺少觀察，努力就沒有方向。

- **C**「發明家都要靠官員的幫忙才能成功」
  學生猶豫的理由：第5課的地鐵確實需要議員批准才能建設，學生可能誤以為「官員同意」是所有發明成功的必要條件。
  錯誤所在：第4課和第6課都沒有提到官員介入；三課的共同主軸是「個人的創意與觀察」，而非外部批准。

- **D**「科技越進步，人類就越不需要動腦思考」
  學生猶豫的理由：第6課強調 QR Code 的便利性（掃一下就知道），學生可能誤讀成「科技讓事情變簡單，就不用思考了」。
  錯誤所在：三課的主旨恰好相反——正是因為人類持續動腦思考、善用創意，才發明出便利的工具；科技進步的前提是更多的思考，而非更少。

## Q29（困難）：連接號「——」的功能

**題目：** 第4課的「春秋時期……有一位著名的工匠——魯班」，句中「——」（連接號）的功能是？

**答案：B**

### 干擾項說明

- **A**「標示時間範圍」
  學生猶豫的理由：連接號有一種用法確實是標示「範圍」（如兩千四百年—兩千八百年），學生可能把數字範圍用法套用到這裡。
  錯誤所在：這裡的連接號接在「著名的工匠」之後，後面是「魯班」，用途是「說明身分」（工匠＝魯班），不是範圍。

- **C**「說明兩千四百年和兩千八百年的差距」
  學生猶豫的理由：題目引文中有「兩千四百年—兩千八百年」，學生可能把括號內的連接號和後面的連接號混淆。
  錯誤所在：題目問的是「——魯班」這個連接號，不是括號內的數字範圍連接號，兩者功能不同。

- **D**「表示接下來的內容和前面無關」
  學生猶豫的理由：學生可能對破折號的用法陌生，猜測這是分隔符號。
  錯誤所在：破折號在此處正是為了「連結」前後文（工匠＝魯班），強調前後有直接的對應關係，而非分隔。

## Q30（困難）：三課發明/發現的共同過程統整

**題目：** 比較三課發明或發現的共同過程，下列哪一項最合理的統整？

**答案：B**

### 干擾項說明

- **A**「三課都描寫主角靠別人提醒才想到解決辦法」
  學生猶豫的理由：皮爾森的地鐵計畫需要議員批准，學生可能誤以為議員批准＝「別人提醒才想到辦法」。
  錯誤所在：三課的靈感都來自主角自身的觀察（草葉、田鼠、生活情境），不是因為別人提醒；議員批准是執行層面，不是靈感來源。

- **C**「三課的主角都是因為意外事故才得到靈感」
  學生猶豫的理由：魯班手被草割傷是意外，學生可能將這個單一事件推廣到三課。
  錯誤所在：皮爾森是長期苦思後偶遇田鼠，QR Code 是觀察生活應用後持續研發——並非「意外事故」；即使是魯班，「手被割傷」是觀察契機，成功的關鍵是他從中得到的思考與行動。

- **D**「三課都強調做發明需要很多錢和大量人力」
  學生猶豫的理由：皮爾森的地鐵確實需要大量費用，議員也質疑過費用問題，學生可能把第5課的個別細節套用到三課。
  錯誤所在：第4課（鋸子）和第6課（QR Code 的生活應用）都沒有強調費用或人力；三課的共同主軸是「個人的觀察→創意→改變生活」，而非資源投入。

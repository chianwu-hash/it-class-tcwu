# Claude Final Review Prompt Template

請以繁體中文回覆。你是這份本地題庫進 Wayground 前的最終審稿者，不是改寫機器。

請用嚴格但務實的標準，判斷這份題庫是否已可進入 Wayground 的 import / check / timer / publish 流程。請不要整份重寫，除非我明確要求。

## 已知任務規格

- 年級：【填入】
- 科目：【填入】
- 教材版本：【填入】
- 範圍：【填入】
- 題數：【填入】
- Bloom 分配：【填入；預設記憶、理解、應用、分析、評鑑平均分配，創造不列入 Wayground 單選預設】
- 題型：Wayground 單選，A-D 四選一
- Wayground 路線：【填入；例如自然 unicode-plus 直接匯入、數學 Wayground AI 出題後清題】

## 請做最後驗收

1. 是否仍有超出範圍或教材不支撐的題目。
2. 是否仍有答案錯誤、答案不唯一或題幹資訊不足。
3. Bloom 分配是否合理，是否過度集中在記憶題。
4. 應用、分析或評鑑題是否真的需要對應思考層次，而不是只靠題幹變長。
5. 干擾項是否有誘答性，尤其是應用、分析、評鑑與高風險題。
6. 是否有重複題、概念過度重複或同一考點換句話重出。
7. 是否有不適合 Wayground 單選的題型。
8. 是否有平台顯示風險，例如自然科化學式、數學式或特殊符號。

## 回覆格式

請分成四段：

### blocking issues

列出必須修正後才能進 Wayground 的問題。若沒有，請寫「未發現阻擋發布的問題」。

### recommended fixes

列出建議修正但不一定阻擋發布的題號與原因。

### residual risks

列出仍需人工最後注意的風險，例如教材版本差異、符號顯示、特定題型轉換。

### final decision

請明確回答：

- 是否建議進入 Wayground import / check / timer / publish 流程
- 若建議，是否需要先修特定題號

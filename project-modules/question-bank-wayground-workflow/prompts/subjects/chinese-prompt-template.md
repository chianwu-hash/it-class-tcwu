# Chinese Subject Prompt Template

在 master prompt 之上，請以指定年級的國語文老師視角命題。

## 本科命題重點

- 可涵蓋字音、字形、詞語、成語、句義、修辭、寫作技巧、課文理解與閱讀理解。
- 若考修辭或寫作技巧，仍要回到學生能從課文或語句中判斷的內容，不可滑成命題後設題。
- 閱讀題應能檢查主旨、人物關係、語句作用、寫作方法或情感表達。
- 若本校紙本考卷常有國字注音、字音字形或詞義辨析，轉成 Wayground 單選時仍要保留能力目標。

## 禁止事項

- 不得用空泛價值判斷取代文本理解。
- 不得只靠選項語感判答案，必須有文本或語文依據。
- 不得因為要做素養題，就整批省略本次範圍要求的基礎語文題型。

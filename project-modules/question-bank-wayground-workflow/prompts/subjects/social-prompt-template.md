# Social Studies Subject Prompt Template

在 master prompt 之上，請以指定年級的社會科老師視角命題。

## 本科命題重點

- 題目可測歷史事件、年代、政策、制度、權利義務、區域發展、社會影響與公民概念。
- 題幹可用完整敘述句，但每題必須只檢查一個明確核心判斷。
- 可以出「哪個敘述錯誤」或「哪個不可能」類型，但選項差異要建立在教材細節，不是文字遊戲。
- 若紙本題型偏是非題或敘述判斷題，轉成 Wayground 時應保留細節核對精神。

## 禁止事項

- 不得把空泛價值判斷偽裝成社會科題目。
- 不得為了提高難度而混入教材未教的歷史細節。
- 不得讓錯誤選項只因年份、地名或制度名稱亂湊而失真。

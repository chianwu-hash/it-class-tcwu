# English Subject Prompt Template

在 master prompt 之上，請以指定年級的英語老師視角命題。

## 本科命題重點

- 可涵蓋發音、字詞、句型、情境理解、短文理解與基本語用。
- 題目應明顯對應學生熟悉的生活主題與教材句型。
- 若題目模擬聽力感，句子要自然、短而清楚。
- 讀寫題要測理解與使用，不要只考文法術語名稱。
- 若紙本題型帶有聽力、配對或看圖性質，請改寫成自然對話題、情境判斷題或短文理解題。

## 禁止事項

- 不得把文法術語辨識當成主要命題方式，除非教材明確要求。
- 不得讓錯誤選項與題幹語境完全無關。
- 不得硬做出需要真實音檔、配對版面或圖片才能成立的失真單選題。

# Distractor Self-Review Gate

這份文件是進 Wayground 前的必要自審門檻，用來避免題庫出現「正答很明顯、錯誤選項很荒謬、困難題只是題幹變長」的問題。

## 1. 什麼時候必做

每份正式題庫進 Wayground 前都要做。

特別是以下題目必須逐題檢查：
- 困難題
- 中等題中具有整合、推論、判斷或計算陷阱的題目
- 國語、社會等容易出現主旨、價值、文本整合判斷的題目
- 數學、自然中需要排除常見錯誤路徑的題目

## 2. 核心問題

對每一道需要檢查的題目，逐一回答：

1. 正確答案為什麼成立？
2. 每個錯誤選項對應哪一種學生可能犯的錯？
3. 哪些錯誤選項會讓「部分理解」的學生猶豫？
4. 有沒有錯誤選項只是荒謬、空泛、亂湊或道德上顯然不好？
5. 正確答案是否因為最長、最完整、最正面、最像標語而太突出？
6. 題幹是否提供足夠資訊，讓學生能靠教材與推理作答，而不是靠猜老師意思？

## 3. 必須重寫的情況

若出現以下任一情況，先修本地題庫，不要進 Wayground：

- 說不出某個錯誤選項為什麼可能被學生選到。
- 三個錯誤選項都明顯荒謬。
- 錯誤選項只是正確答案的粗糙反面。
- 錯誤選項只是在年份、名詞、地名或數字上亂湊。
- 錯誤選項代表教材中沒有任何文本、現象或概念支持的價值立場。
- 正確答案因語氣最完整、最正向或最長而過度顯眼。
- 題目可能有兩個合理答案。
- 困難題的困難只來自題幹更長、用字更繞或敘述更模糊。

## 4. 建議留下的審題紀錄

可在題庫檔案頂端、審題筆記或 Gemini review notes 裡留下簡短紀錄。

格式：

```markdown
## Distractor Self-Review

Q23
- 正答：B，因為...
- A：學生若只注意...，可能誤選。
- C：學生若混淆...，可能誤選。
- D：學生若把...誤解為...，可能誤選。
- 判斷：保留 / 重寫

Q24
- 正答：D，因為...
- A：誘答性不足，重寫為...
- B：可對應常見錯誤...
- C：可對應教材中相近但不相同的概念...
- 判斷：已修正
```

## 5. 最低通過標準

一份題庫要進 Wayground，至少要符合：

- 每一道困難題的三個錯誤選項都能說明誘答來源。
- 多數中等題至少有兩個合理干擾項。
- 沒有荒謬、空泛、後設或只靠道德立場辨識的錯誤選項。
- 沒有正答因形式特徵而過度突出。
- 若有 Gemini 第二審稿，Gemini 對干擾項的疑慮已被採用、修正或明確駁回。

## 6. 與 Gemini 的分工

本流程是本地自審，不可外包給 Gemini。

Gemini 可以協助指出：
- 哪些干擾項太弱
- 哪些選項可能雙答案
- 哪些困難題其實不困難
- 哪些錯誤選項不像學生真實迷思

但最終判斷仍要回到教材、範圍與本地題庫。
